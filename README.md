Data Structure Project (Java)
Overview
This Java console application provides an interactive learning experience with fundamental data structures. Users can explore and experiment with:

Arrays

Linked Lists

Stacks

Queues

Binary Trees

The project offers explanations and allows users to perform basic operations like insertion, deletion, and display to deepen understanding of each data structure.

I made portions of this project with and without using built-in Java classes to showcase my ability to use tools at my disposal whilst also being able to create from scratch.

Features
Array: Create an array with user-defined length, insert integer values, and view contents, built without usiing Java's built-in class.

Linked List: Build a singly linked list with integer insertion and display functionalities, built without using Java's built-in class.

Stack: Demonstrate stack operations (push, pop, display) using Java’s built-in Stack class.

Queue: Explore queue operations (enqueue, dequeue, display) using Java’s Queue interface and LinkedList implementation.

Binary Tree: Interact with a balanced binary search tree using Java’s TreeSet to insert, remove, and display elements.

Project Structure
The entire project is contained in a single main class that offers a menu-driven interface for selecting data structures and performing operations on them. The stack, queue, and binary tree demos utilize Java’s built-in collection classes, while arrays and linked lists are implemented from scratch to illustrate core concepts.

Notes
Designed for educational and demonstration purposes to help users understand fundamental data structures.

Input validation assumes valid integer inputs.

Easily extensible to add more data structures or additional operations.
